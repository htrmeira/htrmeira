How to Create and Attach a New Disk to a KVM Instance

qemu-img create -f raw maas-storage.img 50G

Then, edit the .xml file of the instance:

virsh edit maas

Create a new disk entry:

<disk type='file' device='disk'>
  <driver name='qemu' type='raw'/>
  <source file='/local/virtual-machines/storage/maas-storage.img'/>
  <target dev='vdb' bus='virtio'/>
  <address type='pci' domain='0x0000' bus='0x00' slot='0x07' function='0x0'/>
</disk>

Note that : , the dev must be unique on the file. The same goes to the slot:

Then, restart the virtual machine.

To format the new disk, please look at: http://tldp.org/HOWTO/Partition/fdisk_partitioning.html
